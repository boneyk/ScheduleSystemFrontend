# Фронтенд 2 команды

## Особенности стека
- ⚛️ React 19 с TypeScript
- 🎨 MUI (Material UI) компоненты
- 📦 Webpack 5 с горячей перезагрузкой
- 🔧 TypeScript с поддержкой последних фич
- 🎯 ESLint + Prettier для code quality
- 🎭 SCSS/CSS Modules для стилей
- 🛡 Защищенные маршруты с проверкой прав доступа 
- 🔐 Система аутентификации и роутинга

## Установка зависимостей
```
npm install
```

## Запуск приложения в режиме разработки с hot-reload.
```
npm start
```
- Очищает build директорию
- Создает тег версии
- Запускает линтинг
- Запускает webpack dev server

## Запуск мока
```
npm run mock:auth
```
Сервер запустится на порту 3001

## Доступные endpoint'ы:
Метод	Endpoint	Описание
POST	/auth/login	Вход пользователя
POST	/auth/refresh	Обновление JWT-токенов
POST	/auth/logout	Завершение сессии

## Тестовые данные

Пример тела запроса для /auth/login:
```
{
  "login": "developer",
  "password": "test12345"
}
```

Результаты:

- 200 OK → при верных данных
- 401 Unauthorized → при неверном login или пароле
- 403 Forbidden → при недостаточных правах (если используется правило)
